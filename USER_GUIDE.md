## üß† Memory & Knowledge Management

This is the **secret weapon** of the application. It automatically remembers everything you do and makes it searchable.

### What Is the Memory System?

Think of it like having a **perfect photographic memory** that:
- Automatically saves everything you do
- Organizes it intelligently
- Lets you search by meaning (not just keywords)
- Shows connections between different sessions

### How Auto-Memory Works

**You don't do anything!** The system automatically tracks:
- Video analysis sessions
- Creative Studio generations
- AI chat conversations
- Research activities
- Project work

**Auto-Save Triggers:**
The system saves your session when:
1. You've done **10 activities** (clicks, generates, etc.)
2. **5 minutes** of active work have passed
3. You **close or refresh** the page

**What Happens:**
1. System collects all your activities
2. AI generates a human-readable summary
3. AI creates tags (e.g., "video", "education", "biology")
4. Everything is saved to the cloud
5. You see a subtle notification: **"üíæ Auto-saved to Memory"**

### Using the Memory Tab

#### Step 1: Open Memory Tab
1. Click the **üß† Memory** tab in the AI Panel
2. You'll see two sub-tabs: **Search** and **Graph**

### Search Sub-Tab

#### Semantic Search
This is **not regular keyword search**. It understands **meaning**.

**Example:**
- Search: `"learning about cells"`
- Will find memories about:
  - "biology video on cellular structure"
  - "generated quiz about mitochondria"
  - "research on plant vs animal cells"
- Even if the word "learning" isn't in them!

**How to Search:**
1. Type your query in the search box
2. Press Enter or click üîç
3. Results appear instantly

**Search Tips:**
```
"videos about history"          ‚Üê Find all history videos
"images I generated yesterday"  ‚Üê Find recent images
"research on theology"          ‚Üê Find research sessions
```

#### Filter Results

**By Content Type:**
- üìö Research
- üìù Manual note
- üé• Video analysis
- üé® Creative generation
- üí¨ Conversation

**By Date Range:**
- From: [Select start date]

#### Understanding Results

Each memory card shows:
- **Type icon** (üìöüé•üé®üí¨üìù)
- **Similarity score** (0.95 = perfect match, 0.70 = related)
- **Title** (AI-generated summary)
- **Date** (when it was saved)
- **Preview** (first 200 characters)
- **Tags** (#education #video #biology)

**Similarity Scores:**
- **0.90+** - Exactly what you're looking for
- **0.80-0.89** - Very relevant
- **0.70-0.79** - Related/useful
- **Below 0.70** - Tangentially related

### Graph Sub-Tab

**What Is This?**
A visual **knowledge graph** showing connections between your memories. Think of it like a mind map of everything you've done.

**Why Use It?**
- See connections between different topics
- Discover patterns in your work
- Understand your learning journey
- Find related memories visually

#### How the Graph Works

**Nodes (Circles):**
- Each circle = one saved memory
- **Color** = content type
  - üîµ Blue = Research
  - üî¥ Red = Video
  - üü¢ Green = Creative
  - üü° Yellow = Conversation
  - ‚ö™ White = Manual note

**Links (Lines):**
- Lines connect related memories
- **Thickness** = strength of connection
- Thick line = very related
- Thin line = somewhat related

**Size:**
- Bigger circle = more connections
- Smaller circle = fewer connections

#### Interacting with the Graph

**Navigation:**
- **Scroll** to zoom in/out
- **Click and drag** background to pan
- **Drag a node** to move it (it stays pinned)
- **Double-click a node** to unpin (lets physics move it)

**Hover Effects:**
- **Hover over a node**: Connected nodes and links highlight in gold
- Other nodes fade to 20% opacity
- Easy to see relationships

**Click for Details:**
- **Click any node**: Opens a detailed modal with:
  - Full title and content
  - All tags
  - Creation date
  - Connection count (how many links)
  - Full metadata
  - **Copy button** to copy the content

**Filters:**
- **Content Type**: Show only research, videos, creative, etc.
- **Date Range**: Filter by when memories were created
- Updates graph in real-time

**Export:**
- **PNG**: Save graph as image
- **SVG**: Save as scalable vector (for editing)
- **JSON**: Export data for programming

#### Understanding Connections

**How are memories connected?**
Currently connections are based on **shared tags**. In the future (Days 11-12), connections will also use:
- **Semantic similarity** (similar meaning/topics)
- **Temporal proximity** (created around same time)
- **Combined strength** (weighted algorithm)

**Example:**
If you:
1. Researched "photosynthesis"
2. Watched a video on "plant biology"
3. Generated a quiz about "cells"

The graph will show these three nodes connected because they share tags like #biology, #education, #plants.

### Common Issues

#### "I can't see the AI Panel"
**Solution:**
1. Look for the **ü§ñ** button (usually on the right side)
2. Try refreshing the page (F5)
3. Check if your browser window is wide enough (needs 1200px+)

#### "Video transcript won't load"
**Reasons:**
- Video doesn't have captions/subtitles
- Video is private or age-restricted
- Video is too new (transcripts take time)

**Solution:**
1. Look for videos with the [CC] icon on YouTube
2. Try a different video
3. Use "Auto-generated" captions if available

#### "Image generation is taking forever"
**Normal times:**
- DALL-E 3: 15-30 seconds
- Flux 2 Pro: 20-40 seconds
- Stable Diffusion: 10-20 seconds

**If it's stuck (2+ minutes):**
1. Check your internet connection
2. Refresh the page
3. Try a different model

#### "My memories aren't showing up"
**Checklist:**
1. Are you signed in? (Check top-right corner)
2. Have you done at least 10 activities?
3. Try searching for broader terms
4. Check if you have filters applied (remove them)

#### "The graph isn't displaying"
**Solutions:**
1. Make sure you're in the **Graph** sub-tab (not Search)
2. You need at least 2 memories for connections
3. Try clicking "Type Filter" and selecting "All"
4. Refresh the page

#### "Authentication failed"
**Solution:**
1. Clear your browser cache (Ctrl+Shift+Delete)
2. Try a different browser
3. Try a different sign-in method (Google vs GitHub)
4. Check if cookies are enabled

## üìñ Glossary

**API** - Application Programming Interface. How different parts of software talk to each other.

**Auto-Memory** - The system that automatically saves your work sessions without you clicking anything.

**Claude/GPT** - Names of AI models. Like different versions of smart assistants.

**Consortium** - A group of organizations (or in this case, AI personas) working together.

**DOK** - Depth of Knowledge. A framework with 4 levels measuring thinking complexity.

**Embedding** - A way to turn text into numbers so computers can understand meaning.

**Knowledge Graph** - A visual map showing how different pieces of information are connected.

**LLM** - Large Language Model. The AI technology that powers chat systems.

**Modal** - A pop-up window that appears over the main interface.

**Persona** - A character or personality type. This app has 12 different AI personalities.

**Prompt** - The text you give to AI to tell it what to do.

**Semantic Search** - Searching by meaning, not just keywords. Understands context.

**Synthesis** - Combining multiple sources or perspectives into one coherent summary.

**Token** - A piece of text (roughly 4 characters). AI models charge per token.

**Transcript** - Written text of everything said in a video (captions/subtitles).

## ‚ùì FAQ

**Q: Is this free to use?**
A: The application is free. AI usage costs are paid by the host. Some features (like image generation) have small API costs that may require payment in the future.

**Q: Do I need to install anything?**
A: No! It runs entirely in your web browser. Just visit the website.

**Q: Can I use this offline?**
A: Most features require internet (to call AI APIs), but the interface loads offline after first visit.

**Q: How much data does it store?**
A: Unlimited memories in the cloud (when signed in). Browser cache has ~50MB limit.

**Q: Is my data private?**
A: Yes. Your data is tied to your Google/GitHub account. Only you can see your memories.

**Q: Can I use this on my phone?**
A: Yes! The interface works on mobile, though some features (like the graph) work better on tablets/computers.

**Q: What happens if I don't sign in?**
A: Chat and creative tools still work, but memories won't sync across devices.

**Q: Can I delete my data?**
A: Yes. Memory ‚Üí Settings ‚Üí Delete All Data. Or just sign out.

**Q: Which AI model should I use?**
A: Start with **Sonnet 4.5** (Claude). It's fast, smart, and works great for most tasks.

**Q: How accurate is the AI?**
A: Very accurate for general knowledge, but always verify important facts. AI can occasionally make mistakes.

**Q: Can I use copyrighted YouTube videos?**
A: You can analyze any public video for personal/educational use. Don't redistribute the study materials commercially.

**Q: Why do images sometimes fail to generate?**
A: API rate limits, inappropriate prompts, or service outages. Try a different model or wait a minute.

**Q: How do I report a bug?**
A: Click "Report Issue" in the bottom-right corner, or contact support.
